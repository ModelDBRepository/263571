is available at the authors github repositories:

https://github.com/RonCalabreseLab/HE-model

https://github.com/RonCalabreseLab/HE-model-analysis-matlab

https://github.com/RonCalabreseLab/HE-model-animals-figures

In addition electrophysiology data is available (18GB)

https://datadryad.org/stash/dataset/doi:10.5061/dryad.c0g0p

which they processed with their analysis code

https://github.com/RonCalabreseLab/Burst-Phase-Matlab-lib
